## Generic hardware

These devices can run dahliaOS Linux builds

- **[1]dahliaOS Linux supported:** can run standard dahliaOS builds.

- **[2]dahliaOS Linux-grub supported:** can run dahlia-OS builds but only with the GRUB bootloader.
